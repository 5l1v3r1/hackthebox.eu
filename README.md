# hackthebox.eu
Generate invite codes for Hackthebox.eu

Installation:
```Shell
pip3 install -r requirements.txt
```

Usage:
```Shell
hackthebox.py <numer of invite codes to print>
```
  
Windows:
```Shell
# Print 10 invite codes
py hackthebox.py 10
```

Linux:
```Shell
# Print 10 invite codes
python3 hackthebox.py 10
```
